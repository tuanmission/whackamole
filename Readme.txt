The motivation for this project was to create a simple implementaiton of whack a mole.
To run.

Open index.html

To start the game click 'Start Game'

Moles will randomly appear from their holes.

To score, click a mole and be fast! 

And to end the game click 'End game' 
Moles will stop appearing and your score will be set to zero.

Made in Vanilla JS and HTML CSS
